Blocked List

This page displays all the blocked contacts. You can add the blocked contacts or numbers from

  1. Contacts
  2. Call History
  3. Manual Input which support pattern matching

Call Log

This page displays call history for the contacts or numbers that have been blocked.

Settings

You can perform the followings

  1. Start or stop the SimpleBlocker service. You need to start it in order for it to function.
  2. Call handling behavior, which could be hanging up or silence the call.
  3. Block rules to configure to block all calls, blocked contacts, unknown calls or both unknown and blocked contacts.
